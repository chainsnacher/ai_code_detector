# Power BI Integration Guide

## Overview
This guide explains how to use the Power BI export functionality to visualize and analyze AI code detection results.

## What's Included

The system now automatically exports data to Power BI-compatible formats when you run the training pipeline:

1. **Predictions Data** - CSV file with all prediction results
2. **Model Performance** - CSV with accuracy, F1-score, precision, recall
3. **Feature Importance** - CSV showing which features matter most
4. **Training Statistics** - JSON with dataset statistics
5. **DAX Measures** - Pre-built formulas for Power BI
6. **Setup Instructions** - Step-by-step guide

## Where to Find Exported Data

All Power BI files are exported to: `data/powerbi/`

- `predictions_data.csv` - Prediction results
- `model_performance.csv` - Model metrics
- `feature_importance.csv` - Feature analysis
- `dashboard_predictions.csv` - Dashboard-ready predictions
- `dashboard_performance.csv` - Dashboard-ready performance
- `dashboard_features.csv` - Dashboard-ready features
- `dashboard_stats.json` - Complete statistics
- `measures.json` - DAX formulas
- `POWERBI_INSTRUCTIONS.md` - Setup guide

## Quick Start with Power BI

### 1. Export Data
Run the training pipeline to generate Power BI data:
```bash
python main.py
```

This will automatically create all necessary CSV and JSON files in `data/powerbi/`

### 2. Open Power BI Desktop
1. Launch Power BI Desktop
2. Click "Get Data" → "Text/CSV"
3. Select the files from `data/powerbi/`

### 3. Load Data Sources
Import the following files:
- `dashboard_predictions.csv` 
- `dashboard_performance.csv`
- `dashboard_features.csv`
- `dashboard_stats.json`

### 4. Create Visualizations

#### A. Prediction Distribution
- **Chart Type**: Donut Chart
- **Values**: Count by prediction type
- **Colors**: 
  - Red for "AI Generated"
  - Green for "Human Written"

#### B. Confidence Distribution
- **Chart Type**: Histogram
- **X-Axis**: Confidence (0.0 - 1.0)
- **Y-Axis**: Count of predictions
- **Color By**: Prediction type

#### C. Model Performance Comparison
- **Chart Type**: Bar Chart
- **X-Axis**: Model names (Random Forest, SVM, etc.)
- **Y-Axis**: Accuracy, F1-Score, Precision, Recall

#### D. Language Distribution
- **Chart Type**: Pie Chart
- **Values**: Count by language (Python, Java, JavaScript, etc.)
- **Tooltip**: Show percentage

#### E. Feature Importance
- **Chart Type**: Horizontal Bar Chart
- **X-Axis**: Importance scores
- **Y-Axis**: Feature names
- **Color By**: Feature category

### 5. Add Filters
Create slicers for:
- **Language** (Python, Java, JavaScript, etc.)
- **Date Range** (timestamp filtering)
- **Confidence Threshold** (0.0 - 1.0)
- **Model Name**
- **Prediction Type** (AI vs Human)

### 6. Create DAX Measures
Copy the measures from `measures.json` into Power BI:

```
Total Samples = CALCULATE(COUNTROWS(Predictions))

AI Percentage = DIVIDE(
    COUNTROWS(FILTER(Predictions, Predictions[prediction] = "AI Generated")), 
    COUNTROWS(Predictions)
)

Average Confidence = AVERAGE(Predictions[confidence])

High Confidence Samples = CALCULATE(
    COUNTROWS(Predictions), 
    FILTER(Predictions, Predictions[confidence] >= 0.8)
)
```

## Understanding the Data

### Prediction Labels
- **0 or "Human Written"**: Code written by humans
- **1 or "AI Generated"**: Code generated by AI tools

### Confidence Scores
- **0.0 - 0.5**: Low confidence
- **0.5 - 0.7**: Medium confidence
- **0.7 - 0.9**: High confidence
- **0.9 - 1.0**: Very high confidence

### Language Types
Supported languages include:
- Python
- Java  
- JavaScript
- C++
- C#
- Go
- Rust

### Feature Categories
- **Structural**: AST, tree depth, nesting
- **Lexical**: Tokens, keywords, patterns
- **Documentation**: Comments, docstrings
- **Naming**: Variables, functions, classes
- **Statistical**: Length, size, counts
- **Stylistic**: Indentation, spacing, style

## Troubleshooting

### Missing Data Files
If Power BI files are missing:
```bash
# Re-run the training pipeline
python main.py

# Check if data/powerbi/ directory exists
ls data/powerbi/
```

### Data Not Loading
- Verify CSV file encoding (UTF-8)
- Check for empty files
- Ensure file paths are correct

### Visualizations Not Updating
- Refresh data in Power BI
- Re-import CSV files
- Check for data type mismatches

## Tips for Best Results

2. **Data Cleaning**: Filter out low-confidence predictions
